نوشتن عنوان خیلی راحته. فقط کافیه اولش به مقدار لازم # بذارید:

# عنوان سطح 1

### عنوان سطح 3
#### عنوان سطح 4
##### عنوان سطح 5
###### عنوان سطح 6

بولد کردن و ایتلیک کردن هم به این دو صورت انجام می‌شه:

**بولد**
__بولد__

*ایتالیک*
_ایتالیک_

برای قرار دادن لینک از این روش استفاده کنید:

[عنوان لینک](لینک)

مثال: [شبکه اجتماعی برنامه نویسان](/)

شبکه اجتماعی برنامه نویسان

برای قرار دادن تصویر از این روش استفاده کنید:

![متن جایگزین تصویر](آدرس تصویر)

مثال: ![طبیعت زیبا](http://localhost:8000/cloud/images/yellownature.jpeg)


قرار دادن فهرست هم به همین راحتیه:

- گزینه 1
- گزینه 2
    - زیر مجموعه گزینه 2
- گزینه 3
  • گزینه 1
  • گزینه 2
    • زیر مجموعه گزینه 2
  • گزینه 3
اگه می‌خواین وسط متن یه چیزی رو به شکل کد نمایش بدید از این روش استفاده کنید:

`This is an inline code`

This is an inline code

گر می‌خواید کدی رو تو مطلب قرار بدید از این روش استفاده کنید:

```
x = 1
y = 2
print(x + y)
```

x = 1
y = 2
print(x + y)

در ضمن با قرار دادن نام زبان برنامه نویسی کدها هایلایت می‌شوند:

```python
x = 1
y = 2
print(x + y)
```

اگر هم اسم زبان رو ذکر نکنید با تشخیص خودکار هایلایت می‌شه. اگر کلا هایلایت نمی‌خواید جلوش plain بذارید

x = 1
y = 2
print(x + y)

عنوان 1 عنوان 2
داده ستون 1 داده ستون 2
دومین داده ستون 1 دومین داده ستون 2
